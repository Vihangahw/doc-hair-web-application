our project is to develop a web application called "DOCHAIR" to track hair loss patterns and give solutions in order to prevent from balding once the hair loss type has identified accurately.
The way that the application functions is, once an image has been input into the application the application will go through a dataset of hair-diseases and match it with the disease that it looks most like.
